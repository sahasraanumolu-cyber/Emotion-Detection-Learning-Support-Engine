# 🧠 Emotion Detection & Learning Support Engine

An AI-powered learning assistant that detects a student's emotional state from text and provides personalized learning guidance using a Hybrid Emotion Classification Model and Google Gemini AI.

---

## Features

- Hybrid Emotion Detection (BiLSTM + BERT)
- AI Learning Guidance using Gemini
- CSV Logging
- Session History
- Mixed Emotion Detection
- Learning Analytics Dashboard
- Interactive Plotly Charts
- Confidence Scores
- Dark Mode Streamlit UI

---

## Technologies

- Python
- Streamlit
- TensorFlow
- PyTorch
- Transformers
- Google Gemini API
- Pandas
- Plotly
- Scikit-Learn

